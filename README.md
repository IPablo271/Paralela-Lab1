# Paralela-Lab1

## Instrucciones para compilar

#### piSeriesPar.cpp
```bash
g++ -o piSeriesPar piSeriesPar.cpp
```

#### piSeriesNaive.cpp
```bash
g++ -o piSeriesNaive piSeriesNaive.cpp
```
#### modded.cpp
Para los incisos D, F y G, se modifico el piSeriesNaive y se almaceno en diferentes carpetas bajo el nombre modded.cpp:
```bash
cd Inciso\ D
```
Dentro de la carpeta de su eleccion (D, F, G), se corre el siguiente comando:
```bash
g++ -o modded modded.cpp
```

#### piSeriesAlt.cpp
```bash
g++ -o piSeriesAlt piSeriesAlt.cpp
```
